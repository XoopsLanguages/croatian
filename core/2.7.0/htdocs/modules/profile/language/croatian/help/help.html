OPIS

Modul Profile služi za upravljanje prilagođenim poljima korisničkog profila.

INSTALIRAJ/DEINSTALIRAJ

Nisu potrebne posebne mjere, slijedite standardni postupak instalacije – izvucite mapu /profile u direktorij ../modules. Instalirajte modul kroz Admin -> System Module -> Modules.

Detaljne upute o instaliranju modula dostupne su u XOOPS operativni priručnik

UPUTE ZA UPOTREBU

Za postavljanje ovog modula trebate:

  • Konfigurirajte svoje postavke za modul (pogledajte ‘Postavke’) i po izboru blok profila ako ga namjeravate koristiti (pogledajte ‘Blokovi’).
  • Uredite postojeće kategorije ili dodajte nove.
  • Uredite postojeća polja ili dodajte nova. Ovdje možete odrediti koja će polja biti vidljiva u kojoj kategoriji i hoće li biti vidljiva tijekom registracije korisnika.
  • Definirajte redoslijed koraka registracije.
  • I konačno, možete postaviti dopuštenja za pojedinačna polja - koja se mogu uređivati, a koja pretraživati.

Mjere protiv spama

Kako biste smanjili neželjene registracije, učinite sljedeće:

  • idite na modul Protector u Adminu, idite na Preferences, a zatim na dnu, na ovu opciju: "Stop Forum Spam" Provjerava POST podatke protiv spamera registriranih u bazi podataka www.stopforumspam.com.
    Postavi na "Zabrani IP (bez ograničenja)"

  • u /class/captcha/config.php provjerite je li način rada postavljen kao "text":

    return $config = array(
    'disabled' => false, // Onemogući CAPTCHA
    'mode' => 'text', // default mode, možete odabrati 'text', 'image', 'recaptcha' (zahtijeva api ključ)
    'name' => 'xoopscaptcha', // captcha naziv
    'skipmember' => true, // Preskoči CAPTCHA provjeru za članove
    'maxattempts' => 10, // Maksimalni broj pokušaja za svaku sesiju
    );

  • U modulu Profile idite na Administrator i na Osnovni korak koji se nalazi na:
    /modules/profile/admin/step.php?id=1
    postavite "Spremi nakon zaustavljanja" na "Ne"


  • U postavkama profila postavite opciju "Koristi Captcha nakon drugog koraka registracije" na "Da" (to je zadana)
