Pomoć: Grupe

Opis

XOOPS uključuje sustav registracije članova - po želji možete zahtijevati od ljudi da se prijave na vašu stranicu kako bi imali pristup određenim područjima ili funkcijama. Na primjer, većina web-mjesta zahtijeva od ljudi da se pridruže prije nego što im se dopusti da doprinose vijestima ili postavljaju poruke na forumima.
Pristupna i administrativna prava na web stranici XOOPS postavljaju se putem fleksibilnog sustava dopuštenja temeljenog na 'korisničkim grupama'. Zadane grupe uključuju anonimne korisnike, registrirane članove i webmastere, ali naravno možete definirati onoliko grupa koliko vam je potrebno za upravljanje vašim web mjestom. Svaka grupa može dobiti jedinstven skup prava pristupa koji reguliraju:

- Koji se blokovi mogu vidjeti
- Kojim se modulima može pristupiti
- Koji se moduli mogu administrirati
- Koji se aspekti administracije sustava mogu mijenjati

Možete promijeniti nazive ovih zadanih grupa, ali ih ne možete izbrisati. Ostale korisničke grupe također se mogu postaviti i definirati kao što je objašnjeno kasnije.

Prilagođene grupe također se mogu stvoriti putem stranice za administraciju grupa.
