Instaliranje XOOPS-a

Prva instalacija

Predgovor:

XOOPS je open-source Objektno orijentirani web izdavački sustav napisan u PHP-u. Idealan je alat za razvoj malih do velikih dinamičnih web stranica zajednice, portala unutar tvrtke, korporativnih portala, weblogova i još mnogo toga.

XOOPS je objavljen pod uvjetima GNU Opća javna licenca (GPL) verzija 2 ili novija, te se može besplatno koristiti i mijenjati. Besplatna je redistribucija sve dok se pridržavate uvjeta distribucije GPL-a.

Zahtjevi

  • WWW poslužitelj (Apache, NGINX, IIS, itd.)
  • PHP 8.2 ili noviji, preporučuje se 8.4+
  • MySQL 5.7 .8 ili noviji, preporučuje se MySQL 8.4.5+

Prije instalacije

  1. Ispravno postavite WWW poslužitelj, PHP i poslužitelj baze podataka.
  2. Pripremite bazu podataka za svoju XOOPS stranicu.
  3. Pripremite korisnički račun i dopustite korisniku pristup bazi podataka.
  4. Omogućite upisivanje u direktorije i datoteke: %s
  5. Iz sigurnosnih razloga, preporučujemo da premjestite dva direktorija ispod iz korijen dokumenta i promijenite nazive mapa: %s
  6. Stvorite (ako već nije prisutan) i omogućite upisivanje u direktorije: %s
  7. Uključite cookie i JavaScript u pregledniku.

Instalira se lokalno

Ako koristite lokalno okruženje za razvoj ili testiranje, provjerite imate li prethodno ispunjeni zahtjevi. Kada to učinite, kopirajte sadržaj datoteke htdocs imenik (iz XOOPS distribucijsku datoteku) na stazu korijenskog dokumenta vašeg web okruženja. Nakon što se datoteke tamo kopiraju, možete započeti instalaciju usmjeravanjem preglednika na "http://vasasite.com". Ovo će pokrenuti proces instalacije.

Instaliranje na hostiranu platformu

Ako radite u hostiranom okruženju, raspakirajte XOOPS datoteke lokalno ili na poslužitelju ako imate shell (tj. SSH) pristup. Kada završite, svakako premjestite ili kopirajte sve XOOPS datoteke iz htdocs direktorij u vaš korijenski web direktorij (vaš pružatelj usluga obično daje ovu lokaciju s uputama). Nakon što se datoteke tamo kopiraju, možete započeti instalaciju usmjeravanjem preglednika na "http://vasasite.com". Ovo će pokrenuti proces instalacije.
